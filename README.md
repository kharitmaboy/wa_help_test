# Тестовое задание в компанию Wahelp

Используемый стек:
+ php7.4
+ mysql5.7
+ nginx

Для запуска приложения необходимо выполнить следующие команды:
```
git clone git@github.com:kharitmaboy/wa_help_test.git
make build
make run
make exec-php
php migrate.php
```

Приложение доступно по стандартному порту 8000
